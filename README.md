# MVC-ANDREO

# Objetivo del Proyecto
Desarrollo de una plataforma web de descubrimiento musical que combina las funcionalidades de Letterboxd (reseñas sociales) y Last.fm (seguimiento de escucha), integrando sistemas de recomendación basados en IA y automatización de flujos de trabajo. La plataforma permite a los usuarios explorar álbumes, escribir reseñas, recibir recomendaciones personalizadas y ser notificados sobre actividad relevante en tiempo real.

# Backend: 
PHP 8.1+ con arquitectura MVC personalizada

# Base de Datos: 
MySQL con índices FULLTEXT para búsqueda semántica

# Frontend:
CSS3 personalizado, JavaScript vanilla

# Automatización: 
n8n con contenedores Docker

# Infraestructura: 
Docker Compose para orquestación de servicios

# Servidor Web: 
Nginx con PHP-FPM
